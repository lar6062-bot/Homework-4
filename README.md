# LLM API Interaction & Output Inspection
**Course Assignment** | **Date:** April 22, 2026

## Overview
This project demonstrates how to interact with a Large Language Model (LLM) programmatically using an OpenAI-compatible API. 
## Technical Setup
- **Inference Engine:** LM Studio
- **Model:** NVIDIA Nemotron-3-Nano-4B
- **API Protocol:** OpenAI Python Library
- **Endpoint:** `http://localhost:1234/v1`

## Experimentation & Input Summary
Below are the four questions tested and a summary of the responses generated by the model via the API:

| Question | Model Response Summary |
| :--- | :--- |
| **Golf Shot Shapes** | Identified intentional shapes (Draw/Fade) vs. errors (Slice/Hook) based on ball curvature. |
| **Balance Sheet** | Defined the core accounting equation: **Assets = Liabilities + Equity**. |
| **ChatGPT vs. Gemini** | Provided a comparative analysis of features, training data, and ecosystem integration. |
| **Florida vs. PA Heat** | Explained temperature differences using latitude, solar angle, and oceanic influence. |

## Four Key Findings
1. **Domain Versatility:** The model successfully shifted from technical accounting definitions of aerodynamic ball flight, demonstrating high adaptability across unrelated subjects.
2. **Metadata Inspection:** By monitoring `response.usage`, I observed that the "Climatology" question used the most completion tokens, as the model required a longer explanation to cover latitude and solar radiation.
3. **Contextual Comparison:** When comparing LLMs, the model provided a neutral, structured response, highlighting that API-driven outputs can be consistently formatted for data comparison.
4. **Efficiency of Local Inference:** Using the Nemotron-3-Nano-4B allowed for rapid execution of all four prompts in under 5 seconds, proving that small-parameter models are highly effective for automation tasks.

## How to Run
1. Ensure LM Studio is running with the Nemotron-3-Nano model loaded.
2. Start the Local Server on port `1234`.
3. Install dependencies: `pip install openai`
4. Run the script: `python main.py`
